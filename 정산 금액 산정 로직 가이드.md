# 🧮 로로선 정산 금액 산정 로직 가이드 (Billing Logic Specification)

본 문서는 Node.js 정산 어플리케이션 개발을 위해 **해상 기본 단가에 업체별 마진(Markup) 요율을 적용하여 최종 청구 금액을 산출하는 비즈니스 로직**을 정의합니다.

---

## 1. 정산 산정 흐름도 (Calculation Flow)

1. **기초 데이터 수집**: 화주(업체) 정보, 차량 정보(수량/RT), 당일 고시 환율 가져오기
2. **업체별 매입/매출 단가 확정**: 선사 매입가(Cost)에 업체별 지정 마진율(%) 또는 고정 마진($) 적용
3. **항목별 금액 계산**: 차량 대수 또는 RT를 곱하여 해상 운임 및 로컬 비용 산출
4. **통화 환산 및 합산**: 외화(USD) 항목을 원화(KRW)로 환산 후 로컬 원화 비용과 합산
5. **최종 청구액 산출**: 세금(VAT) 및 최종 올림/버림 처리

---

## 2. 핵심 변수 및 데이터 구조 (Data Structure)

Node.js 백엔드에서 다룰 핵심 객체 구조 예시입니다.

### 2.1 업체별 마진 설정 테이블 (`ClientMargin`)
* 업체마다 마진율을 다르게 설정할 수 있도록 구조화합니다.

```json
{
  "clientId": "DONG_A_TRADE",
  "clientName": "(주)대동자동차무역",
  "marginType": "PERCENTAGE", // PERCENTAGE(정률) 또는 FIXED(정액)
  "oceanFreightMargin": 12.5,   // 해상운임 마진 12.5% 추가
  "localChargeMargin": 10.0,    // 로컬비용 마진 10% 추가
  "fixedMarginPerUnit": 150     // FIXED일 경우 대당 \$150 마진 적용
}
```

### 2.2 선사 원가 및 마진 적용 로직 테이블 (`RateMatrix`)

| 항목 (Charge Item) | 통화 | 선사 매입 단가 (Cost) | 마진 적용 방식 (Billing Rate Logic) |
| :--- | :---: | :--- | :--- |
| **SUV 해상운임** | USD | \$1,600 / 대 | 매입 단가 × (1 + `oceanFreightMargin` / 100) |
| **승용차 해상운임** | USD | \$1,300 / 대 | 매입 단가 × (1 + `oceanFreightMargin` / 100) |
| **고박료 (Lashing)** | KRW | ₩40,000 / 대 | 매입 단가 × (1 + `localChargeMargin` / 100) |
| **터미널이용료 (THC)**| KRW | ₩25,000 / 대 | 매입 단가 × (1 + `localChargeMargin` / 100) |
| **서류비 (B/L Fee)** | KRW | ₩40,000 / 건 | 마진 없이 실비 청구 (Pass-through) |

## 3. 세부 정산 산식 (Mathematical Formulas)

### 3.1 매출 단가 (Selling Rate) 산출
* **정률 마진 (Percentage Markup) 적용 시:**
  $$\text{매출 단가} = \text{매입 단가} \times \left(1 + \frac{\text{마진율}(\%)}{100}\right)$$
* **정액 마진 (Fixed Markup) 적용 시:**
  $$\text{매출 단가} = \text{매입 단가} + \text{고정 마진액}$$

### 3.2 환산 및 최종 청구액 (Total Due) 산출
$$\text{외화 매출 소계 (USD)} = \sum (\text{차량 수량} \times \text{마진 적용된 USD 단가})$$
$$\text{원화 매출 소계 (KRW)} = \sum (\text{차량 수량} \times \text{마진 적용된 KRW 단가}) + \text{고정 로컬비용}$$
$$\text{최종 청구 금액 (KRW)} = \lfloor (\text{외화 매출 소계} \times \text{적용 환율}) + \text{원화 매출 소계} \rfloor \text{ (원단위 절사)}$$

## 4. Node.js 의사 코드 (Pseudo Code) 참조

마크다운 내에 로직 이해를 돕기 위한 백엔드 자바스크립트 계산 함수를 포함합니다.

```javascript
/**
 * 화주별 마진 적용 정산 금액 산출 함수
 * @param {Object} cargo - { suvQty: 3, sedanQty: 2 }
 * @param {Object} margin - 마진 설정 객체
 * @param {number} exchangeRate - 당일 환율 (예: 1350)
 */
function calculateInvoiceAmount(cargo, margin, exchangeRate) {
  // 1. 선사 매입 원가 (Cost)
  const cost = {
    suvOcean: 1600,
    sedanOcean: 1300,
    lashing: 40000,
    thc: 25000,
    blFee: 40000
  };

  // 2. 마진이 반영된 화주 판매가 (Selling Price) 산출
  const oceanMarginRate = 1 + (margin.oceanFreightMargin / 100);
  const localMarginRate = 1 + (margin.localChargeMargin / 100);

  const sell = {
    suvOcean: cost.suvOcean * oceanMarginRate,
    sedanOcean: cost.sedanOcean * oceanMarginRate,
    lashing: cost.lashing * localMarginRate,
    thc: cost.thc * localMarginRate,
    blFee: cost.blFee // 서류비는 마진 제외 실비 청구
  };

  // 3. 항목별 총액 계산
  const totalSuvCount = cargo.suvQty;
  const totalSedanCount = cargo.sedanQty;
  const totalCarCount = totalSuvCount + totalSedanCount;

  // 외화(USD) 합산
  const totalOceanUSD = (sell.suvOcean * totalSuvCount) + (sell.sedanOcean * totalSedanCount);

  // 원화(KRW) 합산
  const totalLocalKRW = (sell.lashing * totalCarCount) + (sell.thc * totalCarCount) + sell.blFee;

  // 4. 환율 적용 및 원단위 절사 (Math.floor)
  const convertedOceanKRW = Math.floor(totalOceanUSD * exchangeRate);
  const finalInvoiceTotal = convertedOceanKRW + totalLocalKRW;

  return {
    detail: { totalOceanUSD, totalLocalKRW, exchangeRate },
    finalAmount: finalInvoiceTotal
  };
}
```
