markdown# 📱 [디자인 가이드] 반응형 레이아웃 및 텍스트 방어 규칙

본 문서는 데스크탑(Table)에서 모바일(Card)로 전환되는 반응형 화면 및 좁은 화면(360px)에서의 레이아웃 깨짐을 방지하기 위한 UI/UX 가이드라인입니다.

---

## 1. 반응형 브레이크포인트 (Breakpoint)

- **Desktop / Tablet:** `768px` 이상
- **Mobile (기본):** `767px` 이하
- **Mobile 최저 해상도 방어 기준:** `360px` (갤럭시 S20 등 대응)

---

## 2. 모바일 반응형 UI 전환 (Table ➡️ Card)

화면 폭이 `767px` 이하로 줄어들 경우, 표(Table) 형태를 유지하지 않고 독립된 **카드 리스트(Card List)** 형태로 컴포넌트를 전환합니다.

### 2.1 정보 다이어트 및 노출 규칙
모바일 카드에서는 데이터의 우선순위를 고려하여 일부 메타 데이터를 숨기거나 노출 형태를 변경합니다.

| 데이터 항목 | 데스크탑 (`>= 768px`) | 모바일 (`<= 767px`) | 개발 적용 예시 (CSS) |
| :--- | :--- | :--- | :--- |
| **대표 타이틀** | 표의 첫 번째 열 | 카드 타이틀 (Bold, 16px) | 기본 유지 |
| **날짜 및 시간** | `YYYY-MM-DD HH:MM` 전체 노출 | `YYYY-MM-DD` (시간 숨김) | `display: none` ➡️ `inline` |
| **상태 값** | 텍스트 형태 노출 | 우측 상단 배지(Badge) 형태 | 레이아웃 재배치 |
| **기타 상세 정보** | 모든 열(Column) 노출 | 기본 숨김 (상세 진입 시 노출) | 리스트에서 제외 |

---

## 3. 텍스트 예외 상황 방어 규칙 (Corner Case)

모바일 화면 폭(최소 `360px`, 양쪽 여백 16px 제외 시 **실제 콘텐츠 폭 약 `328px`**)에서 텍스트가 길어질 때 레이아웃이 깨지지 않도록 아래 규칙을 강제 적용합니다.

### 3.1 말줄임표 (`...`) 자동 처리 (Truncate)
타이틀, 이메일, 주소 등 한 줄을 유지해야 하는 데이터는 영역을 벗어날 경우 다음 줄로 밀리지 않고 말줄임표 처리합니다.

```css
.truncate-text {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
  width: 100%; /* 부모 컨테이너 폭에 맞춤 */
}
```

### 3.2 가변 레이아웃 (Flex Wrap)
라벨과 데이터 값이 가로로 나열될 때, 모바일에서 폭이 부족하면 자동으로 줄바꿈이 일어나도록 설정합니다.

```css
.card-meta-group {
  display: flex;
  flex-wrap: wrap; /* 폭 부족 시 아래로 툭 떨어지게 처리 */
  gap: 8px; /* 항목 간 간격 */
}
```

### 3.3 이미지 및 배지 고정 (Fixed Width)
텍스트가 아무리 길어져도 좌측의 썸네일 이미지나 우측의 상태 배지 크기가 찌그러지지 않도록 고정 폭을 부여합니다.

```css
.status-badge {
  flex-shrink: 0; /* 텍스트가 밀어도 크기가 줄어들지 않음 */
}
```

---

## 4. 모바일 카드 레이아웃 대략적인 구조 (HTML 구조 가이드)

```html
<div class="card-item">
  <div class="card-header">
    <!-- 타이틀 (길어지면 말줄임표) -->
    <h3 class="card-title truncate-text">가장 긴 상품명 또는 제목 텍스트 예시입니다...</h3>
    <!-- 상태 배지 (크기 고정) -->
    <span class="status-badge">완료</span>
  </div>
  
  <div class="card-body card-meta-group">
    <!-- 가변 레이아웃 영역 -->
    <span class="meta-item">등록일: 2026-07-14</span>
    <!-- 모바일에서는 CSS 미디어쿼리로 .time 숨김 처리 -->
    <span class="meta-item time">09:12</span> 
  </div>
</div>
```
