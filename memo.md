# 📝 Antigravity 개발 진행 현황 및 로드맵 메모

## 📅 오늘 완료한 작업 (2026-07-07)

### 1. 차대번호(VIN) 조회 및 상세 제원 UI 개선
- **기본 표시 필터링**: 차량 목록 화면에서 CBM 및 중량(kg)만 기본으로 노출하여 복잡성을 낮춤.
- **상세 제원 접기/펼치기**: 상세 스펙 및 실측 입력 그리드 영역을 접고 펼칠 수 있는 토글 UI 구현.
- **실측값 CBM 자동 계산**: 전장, 전폭, 전고 입력 변경 시 실시간으로 CBM이 재계산되는 로직 추가.
- **VIN 수동 조회 버튼**: 차대번호 입력창 옆에 돋보기(검색) 및 로딩 스피너를 배치하여 VIN Lookup 수동 트리거 지원.
- **야드반입 강등 Confirm**: 야드반입/선적완료 상태에서 다시 운송중/대기중으로 강등할 시 경고창 띄우는 예외처리 적용.

### 2. 화주/포워더 Stepper & 마감 일정 고도화
- **정렬 개선**: Stepper의 연결 선들이 윗배지 크기와 관계없이 동그란 숫자 배지의 정중앙을 세로로 관통하도록 오프셋 정렬 보강.
- **스텝 라벨 및 순서**: 
  - `트럭 운송` ➔ `운송` 라벨로 단어 수정.
  - `선적 완료`와 `해상 운송` 사이에 `출항 (Departed)` 단계 삽입 및 인덱스 동기화.
- **스케줄 마감 시각화**:
  - 서류 업로드 스텝 상단에 `서류마감: MM/DD`, CY 입고 스텝 상단에 `cy입고마감: MM/DD` 표시.
  - 마감 하루 전(D-1)부터 당일(D-Day)까지 해당 배지가 좌우로 횡이동(`animate-wiggle`)하는 경고 모션 탑재.
  - 스텝이 녹색(완료)으로 넘어가면 횡이동이 멈추도록 예외조건 적용.
- **출항 ETD 표시**: `Departed` 단계 위에 실제 ETD 선적 예정 날짜 배지 표시.

### 3. 화주/포워더 스케줄 & 부킹 조건 이관 및 조건 커스텀
- **백엔드 DB 개편**: `bookings` 테이블에 `incoterms` 컬럼을 반영하고 예약 시 저장/조회되도록 업데이트 완료.
- **화주 메뉴 (`SchedulePage`)**: 
  - 필요 부피(CBM) 및 필요 무게(kg) 입력창 복구.
  - 부킹 예약 시 거래 조건(E, F, C, D)을 선택하여 보낼 수 있도록 셀렉트박스 추가.
  - 스페이스(CBM/kg) 잔여량 컬럼은 감추어 화주에게 비노출.
- **포워더 메뉴 (`AdminSchedulePage`)**:
  - 기존 엑셀/MSC 크롤링 폼을 비우고 스케줄 조회 뷰 이식.
  - 부피/무게 검색 조건은 배제하고, 인코텀즈 필드는 `disabled(Read-only)` 상태로 제공.
  - 스페이스 잔여량 컬럼 노출 유지.
- **어드민 부킹 승인판**: `AdminBookingPage` 테이블 내에 화주가 요청 시 선택한 거래 조건(Incoterms) 컬럼을 추가해 승인 전 한눈에 확인 가능하도록 개선.

### 4. 마감일 만료 스케줄 부킹 테스트 지원
- **검색 필터 제거**: 마감일이 지났거나 이미 출항한 과거의 스케줄도 검색될 수 있도록 백엔드 조회 쿼리 일시 수정.
- **부킹 차단 우회**: 백엔드 `requestBooking` 컨트롤러에서 마감 기한 초과 여부를 감시하고 차단하는 예외 블록을 주석 처리하여, 만료된 스케줄로도 테스트 예약을 넣을 수 있도록 우회 처리 완료 (언제든지 복구 가능).
